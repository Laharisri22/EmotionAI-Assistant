# 🧠 EmotionAI Assistant 💖

EmotionAI Assistant is a Machine Learning-based web application that analyzes user emotions from text input and provides intelligent, personalized suggestions to improve mental well-being.

---

## 🌟 Features

✨ Emotion detection from user text  
📊 Intensity prediction  
🧠 Smart decision engine (suggests actions)  
💬 Supportive messages based on mood  
📉 Confidence score visualization  
🌗 Dark / Light mode toggle  
📱 Fully responsive modern UI  

---

## 🧠 How It Works

1. User enters their thoughts in text form
2. Text is cleaned and processed
3. TF-IDF converts text into numerical features
4. Additional inputs are considered:
   - Sleep hours 😴
   - Energy level ⚡
   - Stress level 😣
   - Time of day 🌅
5. Machine Learning models predict:
   - Emotional State
   - Intensity Level
6. Decision Engine suggests actions like:
   - 🫁 Breathing exercises
   - 😴 Rest
   - 📓 Journaling
   - 💻 Deep work
7. A confidence score is shown for prediction reliability
